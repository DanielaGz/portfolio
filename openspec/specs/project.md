# Project: Income API

## Description

This project is a backend that will be used in a dashboard-type frontend for budgeting.

The goal is to create clean and functional code that will serve as the foundation for future features.

## Main features

- Budget CRUD by categories.
- Hexagonal architecture applied to databases.

## Stack

- FastAPI
- Python
- MongoDB

## Limitations

- Do not add unnecessary queries
- Do not add unnecessary libraries
- Keep the code clean and easy to understand

## Project Objective

This project aims to facilitate the storage of users' budget information, which can be consulted as needed.
